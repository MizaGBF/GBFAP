FAQ

• What is this?

This webpage allows the playback of animations featuring characters, outfits, weapons, and main characters from the mobile game Granblue Fantasy.This project is derived from the animation player found within the Chinese Wiki, itself based on the preview of the Gacha page.
This page doesn't host any content. Consequently, it's important to acknowledge that assets could potentially become inaccessible or undergo modifications at the discretion of Cygames.
Keep in mind: Granblue Fantasy content and materials are trademarks and copyrights of Cygames, Inc. or its licensors. All rights reserved.

• Why is the loading so slow?

Because of the existence of Cross-origin resource sharing policies and because no assets are hosted here, javascript and sprite sheet files used for the animations must be fetched via a proxy.
This proxy can take a few minutes to wake up, if it hasn't be used recently.
In doubt, refresh the page if it's still loading after a minute or two.
It also has a limit on the number of hours it can run per month, for cost reasons, so please use this page with parsimony.

• Animations differ between the player and in-game.

In some cases, characters might reuse animations from another.
As such, when an animation is missing, some guesswork is involved, which might lead to disparity between the game and what you see on this page.
Default animations are also used in some cases when no animations at all exist, like for Sakura's auto attack for example.

• When does it get updated?

Updates to this page require manual interventions.
As such, you can expect them to occur between 15:00 and 08:00 JST.
It means that updates won't occur at the time of end-of-month Premium Galas, for example. Please be patient.
In the event that you observe an update to the GitHub repository but no corresponding changes on this page, try a forced refresh through CTRL+F5 or clearing your cache.

• The animation is stuck!

If you noticed a problem or bug on this page, you can open an issue in the GitHub repository or contact me directly.
Try to explain clearly what the problem is and provide screenshots, plus the ID of the element, if possible.

• Class Champion weapons seem to be bugged

For those weapon series, and similar other series, use the first version of the weapon (usually the Fire one) to view the full animations.

• Can you add summons ?

It would require further modifications and I have no plans to add them in the near future.
You are free to make it yourself and open a pull request however, if you have the knowledge to do so, I'll gladly accept.

• What about sounds ?

Same answer. The original preview player isn't intended to play sounds.
Plus, it would require finding out which sound file matches which animation, potentially leading to a colossal amount of work.

• Unreleased asset policy

Concerning unreleased assets, there are instances where Cygames inadvertently makes assets from upcoming content accessible to the general public.
In such occurrences, to prevent any potential complications, you can expect me to promptly delist related elements, if it's apparent that they were intended to remain concealed and if Cygames hasn't already addressed the situation.

• Can I use this project?

The project is open source, under the MIT License.
In the advent that I quit or disappear, you can do whatever you want, within the license scopes, with this project.
Alternatively, you could setup your own local version, if you have the knowledge to do so.

• How can I support you?

You can leave a star on the GitHub repository.
